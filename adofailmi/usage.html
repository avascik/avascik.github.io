adofailmi usage

adofailmi --installumm: installs unitymodmanager, dependencies, and adds additional configuration

(adding -s will skip dependency check and installation, use it when it installing/checking dependencies is broken,

it will not skip steam or adofai checks and it will require manual installation of: mono, whiptail, unzip, wget)

adofailmi --installmods: select and automatically install, unpack mods

adofailmi --createalias: will create the 'umm' alias for easy launching

adofailmi --createsymlink: necessary for installing unitymodmanager

adofailmi --modinfo : will display information about the specified mod

adofailmi --help: displays this page

supported distributions:

- debian/debian based

- ubuntu/ubuntu based

- rhel/rhel based (untested)

- arch/arch based (untested)

- opensuse/opensuse based (untested)

- void linux

- gentoo linux (untested)
